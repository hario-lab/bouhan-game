# 防犯マス取りゲーム（小学生向け）

## 概要
本教材は、小学生向けの防犯教育を目的とした **3×3 のマス取り対戦ゲーム** です。  
赤組と青組に分かれて、防犯に関する問題に答えながらマスを取り合います。

- 正解：自分の色のマスを1つ取得
- 不正解：相手チームに回答権が移動
- 先に3マスを揃えたチームの勝ち

防犯行動をゲーム形式で学べるように設計されています。

---

## 使い方
1. **index.html を開くとゲームメニューが表示されます。**
2. 学年に応じて以下のゲームを選択できます。
   - game_low.html（低学年向け）
   - game_mid.html（中学年向け）
   - game_high.html（高学年向け）
3. ブラウザだけで動作します。インストール不要です。

---

## 問題データの編集（先生向け）
**admin.html** を使うと、問題と選択肢を自由に編集できます。

- 追加
- 削除
- 修正
- 並び替え

編集した問題は **JSON ファイルとして書き出し・読み込みが可能** です。

### 重要（セキュリティ）
- admin.html は **ローカルファイルを編集するツール** です  
- GitHub 上のファイルが書き換わることはありません  
- GitHub Pages 上では編集機能は動作しません（読み取り専用のため）

---
